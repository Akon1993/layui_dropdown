LAYUI · DROPDOWN

一个基于Layui的下拉菜单模块

由个人开发者倾情打造，面向全层次的前后端开发者，低门槛开箱即用的菜单 UI 解决方案

当前版本：2.3.0 更新日志 下载量：16000+

   

简洁的外观
六大亮点
与表格无缝衔接
是时候把你表格里一大堆按钮解放到下拉菜单里了。放心，你将继续有机会在下拉菜单点击后去处理表格的行增删改。
多列菜单
一列菜单已经不够你用了? 没问题，你可以轻松实现两列，三列。只要你喜欢，你甚至可以继续写四五六七列！一次管够！
易于使用
既然选择Layui，你一定熟悉了她的开发习惯以及模块的使用方法。dropdown也延续了这种习惯，让你一度感觉好像就是在使用layui的原生模块。
配置分离
不同的业务提供不同的操作菜单，这些判断冗杂在你的逻辑代码里，使你烦躁了吗？现在你有一个专门判断显示哪些按钮的代码区域了，清洁你的代码！
自定义下拉
可能你并不想用它做下拉菜单。没关系，借助自定义下拉，你还可以下拉任何你喜欢的内容，实现更多个性化的功能。
